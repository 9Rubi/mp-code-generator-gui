## 这是什么?

Mybatis-plus 的 代码生成器 的 **GUI**版

## 如何构建
```cmd
first
    git clone git@github.com:9Rubi/mp-code-generator-gui.git
or  
    git clone https://github.com/9Rubi/mp-code-generator-gui.git
then
    cd mp-code-generator-gui
then
    if u have the maven
        mvn jfx:native
    else
        ./mvnw jfx:native
```
